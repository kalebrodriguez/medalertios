# medalert-app
Medication reminder app built with Swift, featuring scheduled alerts, tracking, and a clean healthcare-focused interface.

# Features:
-  Scheduled medication reminders
-  Real-time notifications
-  Medication Tracking interface

# Tech Stack
-  Swift
-  Xcode

# Purpose
-  This project was built to explore the intersection of technology and healthcare, with a focus on improving patient adherence through simple, reliable design.
